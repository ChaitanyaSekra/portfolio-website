# 🌐 Chaitanya Sekra — Portfolio

## 📁 Project Structure

```
portfolio-website/
├── index.html          # Entry point / redirect
├── home.html           # Landing / hero page
├── about.html          # About me section
├── skills.html         # Technical skills
├── experience.html     # Work & professional experience
├── projects.html       # Portfolio projects showcase
├── certificates.html   # Certifications & achievements
├── loader.html         # Page loader animation
├── script.js           # Shared JavaScript logic
├── styles.css          # Main stylesheet
├── test.css            # Experimental / WIP styles
├── images/             # Image assets
└── sounds/             # Audio assets
```

## 🚀 Getting Started

No build tools required. Just clone and open in a browser.

```bash
# Clone the repository
git clone https://github.com/ChaitanyaSekra/portfolio-website.git

# Navigate into the project
cd portfolio-website

# Open in your browser
open index.html
```

Or use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for a local dev server.
