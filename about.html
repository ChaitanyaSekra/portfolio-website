About Me

I’m Chaitanya Sekra, a developer with a strong interest in AI/ML and software engineering, focused on building systems that actually function beyond demos and tutorials.


I started out as a Web Developer Intern at Dream Soft For You, where I automated workflows using Python and worked with SQL for data handling. Later, as an AI/ML Intern at Data Ingenious, I worked on real-world machine learning projects including face recognition using deep learning embeddings and vector databases, recommendation systems, and transformer-based models. These roles gave me a clear view of how ML concepts translate into production-ready solutions.


My projects reflect this hands-on approach. I’ve built an AI-based Face Recognition System using modern embedding techniques, a Movie Recommendation System using TF-IDF and cosine similarity, an emotion-aware music recommendation system driven by sentiment analysis, and an AI music generator built using transformer models and the Music21 library to generate structured musical sequences. I enjoy working at the intersection of machine learning, data, and software design.


Outside of tech, music is a big part of my life. I write and record rap and original tracks, and that creative process often influences how I think about patterns, structure, and experimentation in my technical work.
